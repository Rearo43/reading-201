# Implementation: Stacks and Queues

### Commons

  - Push: Nodes put into the stack
  - Pop: Nodes removed from the stack 
    > When there is an empty stack an exception should be raised
  - Top: Top of stack
  - Peek: View value of the top Node in the stack
    > When there is an empty stack an exception should be raised
  - IsEmpty: Boolean
